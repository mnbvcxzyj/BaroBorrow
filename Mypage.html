마이페이지

- 내 바로 내역 - 빌린 목록

- 반납 D-DAY를 확인할 수 있고, 반납 상태를 체크할 수 있음
- 반납일이 경과하면 신고버튼이 활성화되고 연체 등의 사유로 빌린 사람을 신고할 수 있음
- 반납 버튼을 클릭하면 확인창이 나와 반납을 완료할 수 있음

장바구니

- 빌리기의 상품 상세 페이지 우측 상단에 위치하는 장바구니 버튼을 클릭하면 장바구니에 상품을 추가할 수 있음
- 장바구니 페이지에서는 사용자가 담은 물건들이 보이고 물건의 정보와 대여 가능 여부를 확인할 수 있음
